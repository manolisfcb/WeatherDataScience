# WeatherDataScience
Complete project of weather analysis and forecasting with python 

In this project we will use:
- nunique() & unique()
- value_counts()
- Filtering
- groupby()
- getgroup()
- isnull() & notnull()
- rename
- mean
- std
- var
- str.contains
- AND (&) and OR (|) operators
- min and max
